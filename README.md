# Бот для записи в салон красоты/барбершоп

Telegram-бот для линейной записи клиента: мастер → услуга → дата → время → контакт → подтверждение. Подходит как для салона красоты, так и для барбершопа.

Бот загружен на [render.com](https://render.com) и доступен по адресу: **[@beauty_salon432_bot](https://t.me/beauty_salon432_bot)**

---

## Возможности

- Запись через пошаговый диалог с инлайн-кнопками
- Свободные слоты только с учетом занятости
- Подтверждение записи с проверкой данных
- Уведомление администратора о новой записи
- Справка по услугам и контакты

## Структура проекта

```
src/
  config.py — настройки и переменные окружения
  db.py — асинхронный движок и сессии
  models.py — ORM-модели: Master, Service, Slot, Booking
  states.py — FSM-состояния диалога
  keyboards.py — инлайн-клавиатуры
  utils.py — выбор свободных слотов
  services/admin_notifier.py — уведомление администратору
  handlers/
    start.py — /start, услуги, контакты
    booking.py — сценарий записи
  main.py — точка входа
scripts/
  seed_demo.py — наполнение демо-данными
```

## Установка

1. Клонируйте репозиторий и установите зависимости:

```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example`:

```bash
copy .env.example .env
```

3. Заполните переменные в `.env`:

```env
BOT_TOKEN=123456:ABC-DEF...
ADMIN_CHAT_ID=123456789
DB_PATH=salon.db
```

- `BOT_TOKEN` — токен бота от @BotFather
- `ADMIN_CHAT_ID` — id чата/пользователя для уведомлений о записях
- `DB_PATH` — путь к SQLite БД

---

## Демо-данные

Для наполнения базы демо-мастерами, услугами и слотами запустите:

```bash
python -m scripts.seed_demo
```

Дамп создастся в указанном в `.env` `DB_PATH` (по умолчанию `salon.db`).

## Сценарий работы

1. Пользователь нажимает «Записаться»
2. Выбирает мастера или «Не важно»
3. Выбирает услугу
4. Выбирает дату
5. Выбирает свободное время
6. Вводит имя и телефон
7. Подтверждает запись
8. Получает подтверждение, администратор получает уведомление

Доступные команды: `/start`, `/cancel` для отмены записи в любой момент.
