# TELEGRAM-БОТ. Запись в салон красоты/барбершоп

## Цель бота
Линейная запись клиента: мастер → услуга → время → подтверждение. Показывает базовый, но аккуратно сделанный сценарий.

## Схема диалога

```
/start
  → Приветствие: «Добро пожаловать в [Название салона]! Помогу записаться на услугу.»
  → Кнопка: [Записаться] [Наши услуги] [Контакты]

[Записаться] →
  Шаг 1: Выбор мастера
    Кнопки: Мастер 1 / Мастер 2 / Мастер 3 / Не важно
  Шаг 2: Выбор услуги (зависит от мастера, если у мастеров разная специализация)
    Кнопки: Стрижка / Окрашивание / Укладка / Маникюр и т.д.
  Шаг 3: Выбор даты
    Кнопки: Сегодня / Завтра / Выбрать дату (следующие 7 дней)
  Шаг 4: Выбор времени
    Кнопки со свободными слотами (например: 10:00, 12:00, 14:00, 16:00)
  Шаг 5: Ввод имени и телефона
    Бот запрашивает текстом: «Как вас зовут?» → «Ваш телефон?»
  Шаг 6: Подтверждение
    Бот показывает сводку: «Мастер: ..., Услуга: ..., Дата: ..., Время: ...» + кнопка [Подтвердить] [Отменить]
  → После подтверждения:
    - Клиенту: «Вы записаны! Ждём вас [дата, время]. Адрес: ...»
    - Админу/мастеру: уведомление в личку/отдельный чат с деталями записи

[Наши услуги] →
  Список услуг с ценами (текстовое сообщение или карточки)

[Контакты] →
  Адрес, телефон, часы работы, кнопка «Открыть карту» (deep link на Яндекс/Google карты)
```

## Технические детали
- Реализация: инлайн-кнопки (InlineKeyboardMarkup) для выбора шагов, чтобы не засорять чат
- Хранение записей: простая таблица (Google Sheets через API или локальная БД SQLite) — мастер, услуга, дата, время, имя, телефон
- Проверка занятости слотов: при выборе времени бот должен исключать уже забронированные слоты
- Уведомление администратору — либо в отдельный Telegram-чат/канал, либо личным сообщением

## Демо-данные для портфолио
- Название: «Barber Studio» или аналог
- 2-3 мастера с вымышленными именами
- 4-5 услуг с ценами
- Слоты на ближайшие 3-5 дней (можно захардкодить для демонстрации)

## Подача в портфолио
Запишите короткий скринкаст (10-15 сек, экран телефона) — GIF или видео прохождения сценария целиком, от /start до подтверждения записи.
